# 💅 Manicure Booking Bot

Telegram-бот для записи клиентов к мастеру маникюра.

---

## 📁 Структура проекта

```
manicure_bot/
├── bot.py                  # Точка входа
├── config.py               # Настройки (токен, ID, канал)
├── requirements.txt
├── database/
│   ├── __init__.py
│   ├── db.py               # Инициализация SQLite
│   └── queries.py          # Все SQL-запросы
├── handlers/
│   ├── __init__.py
│   ├── common.py           # /start, прайс, портфолио
│   ├── booking.py          # FSM записи клиента
│   └── admin.py            # Административная панель
└── utils/
    ├── __init__.py
    ├── calendar_kb.py      # Inline-календарь
    └── scheduler.py        # APScheduler напоминания
```

## ⚙️ Установка и запуск

### 1. Клонируйте / скопируйте проект

```bash
cd manicure_bot
```

### 2. Создайте виртуальное окружение

```bash
python -m venv venv
source venv/bin/activate       # Linux / macOS
venv\Scripts\activate          # Windows
```

### 3. Установите зависимости

```bash
pip install -r requirements.txt
```

### 4. Настройте `config.py`

| Параметр | Описание |
|----------|----------|
| `BOT_TOKEN` | Токен от @BotFather |
| `ADMIN_ID` | Ваш Telegram ID (узнать у @userinfobot) |
| `SCHEDULE_CHANNEL_ID` | Username канала расписания, например `@my_schedule` |
| `CHANNEL_ID` | Username канала обязательной подписки |
| `CHANNEL_LINK` | Ссылка на канал `https://t.me/...` |

> ⚠️ Бот должен быть **администратором** в каналах `SCHEDULE_CHANNEL_ID` и `CHANNEL_ID`.

### 5. Запустите бота

```bash
python bot.py
```

## 🔧 Использование

### Для клиентов
- `/start` — главное меню
- **Записаться** — выбор даты и времени через inline-календарь
- **Отменить запись** — отмена активной записи
- **Прайсы** — список услуг и цен
- **Портфолио** — ссылка на Pinterest

### Для администратора
- `/admin` — открыть панель управления
  - Добавить рабочий день (создаёт день + слоты по умолчанию)
  - Добавить/удалить временной слот
  - Закрыть/открыть день
  - Просмотреть расписание на дату
  - Отменить запись клиента

---

## 📌 Особенности

- Один пользователь — одна активная запись
- При отмене слот автоматически освобождается
- Напоминание отправляется за 24 ч (не создаётся, если < 24 ч до визита)
- Напоминания восстанавливаются после перезапуска бота
- Обязательная проверка подписки перед записью
